HTML Básico

Aprenda os fundamentos do HTML e dê os primeiros passos no desenvolvimento web. Neste curso, você vai criar sua primeira página index.html, entendendo a estrutura de um site, uso de textos, imagens, links e boas práticas de organização do código. Ideal para iniciantes que querem começar do zero.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML. Com ele, é possível definir cores, fontes, espaçamentos, tamanhos e layouts, deixando o site mais bonito, organizado e responsivo.

JavaScript

JavaScript é a linguagem responsável por tornar as páginas web interativas e dinâmicas. Com ela, é possível criar ações como cliques, animações, validação de formulários e manipulação de conteúdos em tempo real.
